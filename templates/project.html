TO DO APP

This app will help the user to keep track of the tasks and can be able to add as many tasks as possible and delete the task after it is done.

1. User can add the task in a descriptive way.

2. User can add as many tasks as the want.

3. They can edit the task by swiping right the task.

4. They can delete the task by swiping left.

5. When the task is completed, user can tick the tasks in the checkbox.

6. The tasks are stored in a list view.

PREREQUISITES OF THE PROJECT

1. Java programming

2. Extensible markup language

3. Android studio
